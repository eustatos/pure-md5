# Задача 1: Создание базового класса MD5Stream

## Цель
Создать базовый класс `MD5Stream`, расширяющий `stream.Transform` для потокового вычисления MD5 хэша.

## Требования к реализации
1. **SRP**: Класс должен отвечать только за вычисление MD5 хэша в потоковом режиме
2. **Производительность**: Минимальные накладные расходы на обработку каждого чанка
3. **Совместимость**: Работа с любыми потоками Node.js (Readable, Writable, Transform)

## Детали реализации
- Наследование от `stream.Transform`
- Инициализация MD5 хэша в конструкторе
- Обновление хэша в методе `_transform`
- Финализация хэша в методе `_flush`
- Эмит события с результатом хэширования

## Критерии выполнения
- [ ] Создан класс `MD5Stream` в файле `src/stream/md5-stream.js`
- [ ] Класс корректно расширяет `stream.Transform`

- [ ] Реализован метод `_flush` для финализации хэша
